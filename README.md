## Project Overview
This project provides a comprehensive analysis of DMart's sales performance, customer demographics, and logistics efficiency. The dashboard transforms raw transactional data into actionable insights, helping stakeholders identify high-performing regions and optimize inventory based on product category demand.

## Key Insights & Features
* **Revenue Analysis:** Tracking total sales (₹23.01M) and profit (₹11.16M) across different timeframes.
* **Geographic Distribution:** Visualizing sales concentration across states like Maharashtra, Telangana, and Andhra Pradesh.
* **Customer Segmentation:** Analysis of sales by Gender and Age groups to understand the primary consumer base.
* **Logistics Performance:** Monitoring order status (Shipped, Cancelled, Pending) and shipping mode preferences.
* **Product Performance:** Identifying top-selling categories (e.g., Branded, Local, Imported).
